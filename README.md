# **Pizza Sales Dashboard**

## **Description**
This Power BI dashboard provides an analysis of pizza sales data, focusing on key metrics like total orders, total sales, average order value, and quantity sold. It also breaks down sales performance by category and time of day/day of the week.

## **Key Performance Indicators (KPIs)**
* **Total Orders:** 21.4K
* **Total Sales:** 817.9K
* **Avg Order Value:** 38.3
* **Total Pizza Quantity:** 49.6K

## **Filter Panel**
The dashboard allows dynamic filtering based on:
* **Weekdays:** (Currently set to 'All')
* **Pizza Category:** (Currently set to 'All')

## **Quantity by Pizza Size**
The distribution of pizza quantity sold by size is heavily skewed towards larger sizes:
1.  **L (Large):** 19.0K
2.  **M (Medium):** 15.6K
3.  **S (Small):** 14.4K
4.  **XL (Extra Large):** 0.6K
5.  **XXL (Double Extra Large):** 0.0K

## **% of Sales by Category**
Sales are well-distributed across the main categories:
1.  **Classic:** 26.91%
2.  **Supreme:** 25.46%
3.  **Chicken:** 23.96%
4.  **Veggie:** 23.68%

## **Sales by Weekday and Time of Day (Hourly)**
The table provides detailed sales figures, highlighting peak times:
* **Peak Day:** Wednesday shows the highest sales (e.g., 17,127.55 at 13:00).
* **Peak Hour:** The **13:00 (1 PM)** slot consistently shows high sales across all weekdays.
* **Lowest Sales:** The **10:00 AM** slot consistently shows the lowest sales.

## **Technologies Used**
* **Power BI Desktop**
* **DAX** for calculated columns and measures
* **Power Query** for data transformation
